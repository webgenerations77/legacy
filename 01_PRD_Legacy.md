# Legacy - Product Requirements Document (PRD)

## Vision
Legacy is an AI-powered life organization and legacy planning platform that helps individuals organize personal, financial, legal, medical, and digital information, and guides families through execution during crisis or after death.

## Core Users
- Primary User (planner)
- Trusted Contacts
- Survivors (post-activation users)
- Admin (system operators, no vault access)

## Core Modules
- Personal Profile
- Financial Accounts
- Bills & Subscriptions
- Legal Documents Vault
- Digital Accounts Manager
- Medical & Emergency Planning
- Funeral Planning
- Obituary Generator
- Memory Vault
- Letters & Messages for Future
- Survivor Activation System
- AI Legacy Assistant

## Key Differentiator
AI-first interface that allows conversational updates instead of form-based entry.

## MVP Scope
- Auth + encryption
- Profile + documents vault
- Financial & bill tracking
- Basic AI assistant
- Survivor checklist system
