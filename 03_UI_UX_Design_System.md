# UI/UX Design System

## Design Philosophy
Calm, human, non-legal, non-financial feeling. Apple-like simplicity.

## Color Palette
- Soft neutrals
- Warm grays
- Subtle blues
- No harsh reds except alerts

## Layout Principles
- Conversational UI first
- Minimal dashboards
- Progressive disclosure

## Core Screens
- Home Assistant Feed
- Legacy Completion Score
- Vault
- Family Access Hub
- Survivor Mode Dashboard

## Tone
Supportive, calm, non-alarming
